# bbb-permeability-model
This repository aims to develop an AI model that can predict the permeability of a specific compound across the blood brain barrier (BBB) 

## Structure of Repository 
The repository is organized into different subcategories. These subcategories include: 
- BBB Background 
- Paper Summaries 
- Source Code 

### Subcategory: BBB Background 
BBB Background contains markdown files relating to the explanation of the blood brain barrier's structure and function. 

### Subcategory: Paper Summaries
Paper Summaries contains summaries of papers relating to the structure and function of the blood brain barrier, as well as past attempts to develop predictive AI models and their respective methodology 

### Subcategory: Source Code
Source code contains the code for data curation, model development, and model training.
